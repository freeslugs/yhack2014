yhack2014
=========

## install
1. clone repo
  1. `git clone https://github.com/freeslugs/yhack2014/`
  1. `cd yhack2014`
1. set up virtualenv
  1. `pip install virtualenv`
  1. `virtualenv venv`
1. active virtual env
  1. `source venv/bin/activate`
1. install requirements
  1. `pip install -r requirements.txt`
